## [1.0.0]
### Added
- Area and window screenshot capture with a global hotkey each.
- Screen recording: record an area or a window to a `.mov`, each with its own hotkey.
- Corner overlay with copy / reveal / pin and drag-out; hides while dragging.
- Inline discard: tap ✕ to reveal a red trash, tap ✕ again to keep, or the trash to delete.
- Pin-to-screen floating windows (move, scroll-resize, opacity, Esc to close).
- Onboarding + Settings: customizable shortcuts, save folder, capture sound, launch-at-login.
- Selectable capture sounds (Bubble / Blip / Clack / Shutter / System Default / None) with previews.
- Menu bar version display and a Help link to the GitHub project.
- Local, no-cloud, no-network by design.
